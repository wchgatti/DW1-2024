Aqui está um exemplo de uso da tag <code>:

function greet() { console.log("Olá, mundo!"); } greet();
Neste exemplo, a função greet() em JavaScript está envolta pela tag , fazendo com que seja exibida em uma fonte monoespaçada, facilitando a identificação como código. Além disso, caracteres especiais como '<' e '>' são automaticamente convertidos em entidades HTML, como '& l t' e '& g t' , para que sejam exibidos corretamente no navegador.

Aqui está um exemplo de uso da tag <samp>:

Olá Mundo! Este é um exemplo de saída.
Neste exemplo, a tag <samp> é usada dentro da tag <code> para representar a saída de um programa. O navegador irá exibir o texto dentro da tag <samp> em uma fonte monoespaçada, sem interpretar o conteúdo como código HTML, o que é apropriado para exibir saída de programa.
